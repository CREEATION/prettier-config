Repository holding my personal Prettier configuration.

## Installation

```console
$ npm i -D @creeation/prettier-config
```

## Usage

In your `package.json`, set `prettier` to `@creeation/prettier-config`:

```json
{
  "prettier": "@creeation/prettier-config"
}
```
